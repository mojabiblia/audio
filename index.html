Biblia Audio została skopiowana ze strony
http://biblia-online.pl/BibliaAudio/Warszawska
a tam udostępniona dzięki życzliwości https://ccnz.pl/

Klikając na dany rozdział zostanie on ściągnięty lokalnie w formacie mp3.
Na system Android polecamy odtwarzacz AIMP który umożliwia wygodne odtwarzanie plików mp3 z możliwością przyspieszenia prędkości odtwarzania.
( https://play.google.com/store/apps/details?id=com.aimp.player )

Aby ściągnąć wszystkie pliki naraz należy zainstalować małą aplikację GIT
(https://git-scm.com/downloads), a następnie wpisać komendę:
  git clone https://github.com/mojabiblia/audio.git
w wierszu poleceń w wybranym katalogu lokalnym. To wykona kopię całego repozytorium ze wszystkimi plikami mp3 z Biblią Audio.
